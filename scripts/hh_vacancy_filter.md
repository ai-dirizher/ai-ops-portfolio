# HH Vacancy Filter

Сканер вакансий HH с нишевым фильтром через CDP.

## Что делает

1. Подключается к persistent Chrome на порту 9223
2. Извлекает вакансии с текущей страницы HH
3. Классифицирует их по нишевому фильтру

## Фильтр

**OUT (пропускаем):**
- Телефонные звонки, обзвоны
- Reels / видео / монтаж
- Создание/генерация картинок
- Sales / MLM
- "За лайки" / "за отзывы" / микрозадачи
- "Возраст от 14-16" / "без опыта подходит всем"
- Резюме-блокеры (Авито)

**IN (откликаемся):**
- AI / ML / LLM
- Автоматизация / n8n / pipeline
- Аналитика / SQL / Data Science
- Маркетплейсы (WB, Ozon)
- Копирайтинг / редактура
- SMM (текстовый, без Reels)
- Telegram-боты
- Ассистентские роли

## Использование

```bash
pip install websocket-client
python3 hh_vacancy_filter.py
```

Перед запуском:
1. Откройте `https://nn.hh.ru/search/vacancy?work_format=REMOTE` (или ваш URL с фильтром) в persistent Chrome
2. Запустите скрипт — он извлечёт и классифицирует вакансии

```
=== IN (3) ===

    FL Haus  ::  ai, smm
    https://nn.hh.ru/vacancy/...

=== OUT (12) ===
  [от 49 000 ₽] Оператор контактного центра
    ВТБ  ::  calls
    https://nn.hh.ru/vacancy/...
```

## Заметки

- Требует persistent Chrome с `--remote-debugging-port=9223`
- Использует `data-qa` атрибуты HH (стабильные селекторы)
- Не делает отклики — только классифицирует. Для откликов используйте `hh_apply.py`
