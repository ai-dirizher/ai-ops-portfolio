## Кто я

AI-оператор и автоматизатор. Не «делаю ботов» — ставлю AI-агент (Hermes, N8N) в рабочий процесс и закрываю им рутину, которая раньше требовала живых людей.

Работаю с AI-агентами с момента их появления. Сильные стороны: быстро разбираюсь в новых инструментах, довожу задачи до конца, не срываю сроки, не исчезаю.

📍 Нижний Новгород · Работаю удалённо

## Кейсы

### AI-агент для личного архива Obsidian

Полноценный архив знаний, который ведётся AI-агентом автономно:
- 23+ категории картинок, рассортированных по темам
- Чанкинг книг — автоматическая разбивка и индексация
- Ежедневное ведение структуры — без моего участия
- Поиск по архиву через AI

Стек: Hermes Agent, Obsidian, Markdown, Python.

### Workflow 350+ нод в N8N для автопубликации контента

Единый workflow, который принимает контент и публикует его на 9-13 разных площадок в зависимости от типа:
- Текст → Telegra.ph, VK, Telegram-каналы
- Картинки → Pinterest, VK, Imgur, Reddit
- Видео → YouTube Shorts, TikTok, VK Clips

Стек: N8N, HTTP API, OAuth, webhook-и.

### Telegram-бот с интеграцией LLM (Hermes Agent)

Полноценный AI-ассистент «на все случаи жизни»:
- Юридический помощник
- Финансовый консультант
- Поисковик по интернету
- Рекомендатель фильмов
- Память контекста между сессиями

Стек: Hermes Agent, Python, Telegram Bot API, RAG.

### Торговые боты

Боты для криптобирж с автоматическим выполнением стратегий.

Стек: Python, ccxt, API бирж.

### Парсинг данных

Парсинг сайтов с обходом защит, капчи, пагинации.

Стек: Python, BeautifulSoup, Selenium, Playwright.

## Что я делаю как AI-оператор

- Настраиваю AI-агентов для бизнеса (Hermes, N8N, OpenAI API)
- Пишу Telegram-ботов с интеграцией LLM
- Автоматизирую рутину (парсинг, отчёты, оповещения)
- Делаю email-воронки и авто-рассылки
- Строю workflow для маркетплейсов (WB, Ozon)
- Веду Obsidian-архивы знаний
- Пишу торговых ботов
